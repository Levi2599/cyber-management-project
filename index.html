🔗 חיבור אוטומטי ל-Repository (Layer B)

במקום להעתיק ולהדביק קטעי קוד, המערכת מתחברת ישירות למאגר ה-GitHub שלך באמצעות Webhooks. בכל פעם שתפתח Pull Request או תעשה Push, המערכת תסרוק את השינויים (ה-Diff) אוטומטית, תכתוב הערת אבטחה ב-PR ותדחוף תיקון אקטיבי ישירות לענף שלך.

  • 1 העתק את כתובת ה-Webhook שלך מ-n8n:
    http://localhost:5678/webhook/github-webhook
  • 2 הגדר את ה-Webhook ב-GitHub:
    גש למאגר שלך ב-GitHub $\rightarrow$ Settings $\rightarrow$ Webhooks $\rightarrow$ Add Webhook. הדבק את הכתובת שהעתקת, הגדר את ה-Content type ל-application/json ובחר להאזין לאירועי Pull Requests ו-Pushes.
  • 3 סריקה אוטומטית ברקע:
    זהו זה! החיבור הושלם. כעת כשיווצר שינוי בקוד במאגר, סוכני ה-Red & Blue Team יפעלו ברקע ויבצעו את התיקון ישירות בתוך GitHub.
